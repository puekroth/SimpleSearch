## Create JAR file in Eclipse: 
1. Open the project in Eclipse
2. Click SimpleSearch -> Export -> Java -> Runnable JAR file

=============================================================

## Run JAR file:
1. Open a command prompt
2. Change to the directory of the JAR file
3. Run with: java -jar SimpleSearch.jar /foo/bar (given that /foo/bar is the directory of the text files to search)

=============================================================

## Run JUnit test:
1. Download the /foo/bar folder and put in default directory
2. Open the project in Eclipse
3. Open SimpleSearch -> src -> main -> Test.java
4. Run Test
